How to create match in Slendertubbies

This document discusses all the steps of creating a match in Slendertubbies.

First of all in main menu you have to press "Start Game" button. Now I'll explain how this menu works. We are going to skip networking elements, because I'll explain them in "How to play online" tab. Default game setup will open Singleplayer match.

  • Game modes

    Slendertubbies has 2 different game modes. All of them are available in singleplayer and multiplayer modes.

    • Collect

      This is a classic horror game mode. The player must collect all the custards in order to win the game. The player takes the role of the victim. The time to collect the creams is unlimited. If you are caught by teletubby, you will lose.

    • Versus

      Versus game mode is a bit more complicated. The host player takes on the role of Teletubby. The goal of the Teletubby player is to eliminate all survivors before they collect all custards. The survivors must collect all the custards and not let the Teletubbies player touch them. Teletubby will win when all players will be catched. Survivors will win when all custards will be collected. This is strictly a multiplayer game mode and works best with real players. However, you can play it in singleplayer mode with bots.

  • Maps

    Slendertubbies has 8 different maps. All of them are available in singleplayer and multiplayer modes. Map names are pretty much self-describing.

    • Woods
    • Woods (Legacy)
    • Maze
    • Lake
    • Bunker
    • Tubbycamp
    • Castle
    • Swamp Wastelands

    I think that best maps are: Bunker, Swamp, Castle and maze.

  • smount of custards

    This is pretty much self-describing. You can specify how many custards will be spawned.

    Default is 10.

  • Rules

    Rules are pretty cool Slendertubbies feature. As you see game has only 3 of them. They can make game harder.

    • Select tubby

      This rule allow you to select teletubby. It controls which Slendertubby will be spawned. In Versus game mode this option will allow player to select versus teletubby. Map Maze has this option is always enabled.

    • Flashlight madness

      This rule ads battery consumption of the game. When this game rule is active, the player's flashlight loses batteries. To keep the flashlight running, you must collect batteries.

      This game rule works in all game modes, and both in singleplayer and multiplayer.

    • Full tubby jacket

      This rule Spawns every possible Slendertubby into game. This game mode makes game really hard.

      This game rule works only in collect, and both in singleplayer and multiplayer.

  • Accept

    This button starts game.
