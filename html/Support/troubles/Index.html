Troubleshooting

Currently Slendertubbies shouldn't have major issues with launching.

If you are installing game from Itch app, you shouldn't have any redistributable issues. If you are running on your's pc first time game made in Unreal Engine, game should automatically open engine redistruables setup. The same with .Net, game will open setup window automaticaly.

If you have troubles with launching game visit discord server.
