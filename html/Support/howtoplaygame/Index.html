How to play Slendertubbies

This document will introduce you to the gameplay of Slendertubbies

Let's talk about how to play Slendertubbies.

Controls

  • Slendertubbies has basic keyboard controls. They will be listed below.

    • WASD - This controls are responsible for players movement.
    • Right mouse button/Left Shift - This controls are responsible for Sprint.
    • Space - is responsible for jump.
    • C - shows remaining custards.
    • Z - toggles Sprint.
    • I - toggles in-game UI.
    • H - toggles in-game key binds hints.
    • P/Esc - toggles pause menu.

  • Multiplayer

    • TAB - shows currently connected players.
    • M - opens player management menu.
    • T - opens chat.

Autoplayers

  • Slendertubbies has basic bots that acts as players. They allow you to play offline Versus game mode. Here I'll show You how to spawn them and if there is such need how to remove them.

    • Spawning Autoplayes

      In order to spawn AI player you have to make sure that you are running game with Versus game mode enabled. You have to be game host!

      Press in-game "M" key. You will see this menu.

      Below I'll describe how this menu works.

      1. This button closes this menu.
      2. This button adds new Autoplayer.
      3. This button set AI difficulty to easy.
      4. This button set AI difficulty to hard.

      You can spawn new Autoplayer by pressing button with number 2.

      There is no limit how many you can spawn them. Maximum amount of bots depends on your hardware spec. I recommend to play with 4-6 bots.

      Bots has 2 difficult levels. You can specify difficult level by using buttons 3 and 4. On hard difficulty bots know maps better and can spot you from greater distance.

    • Despawning Autoplayes

      Each bot and player has its own row in this menu.

      Bots has [AI] next to theirs nick. You can kick them by just pressing 'kick' button.
